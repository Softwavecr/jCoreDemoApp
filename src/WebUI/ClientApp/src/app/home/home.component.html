Hello, jVisitor!

Welcome to your new single-page application, built with:

  • ASP.NET Core and C# for cross-platform server-side code
  • Angular and TypeScript for client-side code
  • Bootstrap for layout and styling
  • OpenWeather API for LIVE Weather Data
  • In Memory Test DB for immediate functionality without external DB dependencies
  • Swagger can help you design and document your APIs at scale
  • Clean Architecture Solution Template clearly separates concerns amongst Domain, Application, Presentation and Infrastructure layers

The ClientApp subdirectory is a standard Angular CLI application.

If you open a command prompt in that directory, you can run any ng command (e.g., ng test), or use npm to install extra packages into it.
